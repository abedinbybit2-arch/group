📱 Telegram Account Login

আপনার নিজের Telegram account দিয়ে লগইন করুন — account-এর সব group দেখুন এবং এক ক্লিকে সবগুলোতে bot add করুন

⚠️ এটি Telegram-এর MTProto API দিয়ে সরাসরি browser থেকে আপনার আসল Telegram account-এ লগইন করে (bot token নয় — real personal account)। Login session এই browser-এর localStorage-এই থাকে, Firebase-এ পাঠানো হয় না।

অনেক group-এ খুব দ্রুত bot add করলে Telegram সেটাকে spam/abuse মনে করে account সাময়িকভাবে সীমিত করে দিতে পারে — তাই প্রতি group-এর মাঝে delay দেওয়া আছে এবং একসাথে অনেক (৫০+) group না করে ধীরে ধীরে করার পরামর্শ দেওয়া হচ্ছে।

AI Auto-Reply (Google Gemini)

Group এ message আসলে Google AI Studio (Gemini) দিয়ে automatic reply পাঠানো হবে — লাইভ ওয়েব সার্চ ও থিংকিং সহ, সবচেয়ে ভালো উত্তরের জন্য

⚠️ Bot group-এর সব message দেখতে পারার জন্য @BotFather-এ গিয়ে /setprivacy কমান্ড দিয়ে bot-এর Privacy Mode Disable করুন, অথবা bot-কে group-এ Admin বানান — নাহলে bot শুধু mention/reply করা message-ই দেখতে পাবে।

এই ফিচারটি এই ব্রাউজার ট্যাব খোলা থাকা পর্যন্তই active থাকবে (এটি একটি client-side polling loop, server-side নয়) — ট্যাব বন্ধ করলে বা পেজ রিলোড দিলে Auto-Reply বন্ধ হয়ে যাবে এবং পুনরায় "চালু করুন" চাপতে হবে।

🔑 Google AI Studio API Settings

🔗 ফ্রি API Key পাবেন: aistudio.google.com/apikey (Google AI Studio)

🧠 প্রতিটি reply-তে স্বয়ংক্রিয়ভাবে Thinking (গভীরভাবে ভেবে উত্তর তৈরি) এবং Google Search Grounding (লাইভ ইন্টারনেট থেকে আপ-টু-ডেট তথ্য খুঁজে আনা) চালু থাকবে — তাই উত্তর সবসময় সর্বোচ্চ মানের ও বর্তমান তথ্য-ভিত্তিক হবে।
